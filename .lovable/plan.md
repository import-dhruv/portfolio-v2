

# AI Engineer Portfolio — Dark & Techy

## Design
- **Dark theme** with deep navy/charcoal background, neon cyan/purple gradient accents
- Code-inspired elements: monospace fonts for headings, terminal-style decorations, subtle grid/dot patterns
- Smooth scroll animations and hover effects throughout
- Fully responsive

## Sections

### 1. Hero
- Large name + "AI Engineer" title with gradient text effect
- Animated typing effect for taglines (e.g., "Building intelligent systems", "Machine Learning • NLP • Computer Vision")
- CTA buttons: "View Projects" and "Contact Me"
- Subtle particle/grid background animation

### 2. About
- Brief bio with a placeholder avatar
- Key stats (years of experience, projects completed, etc.)
- Terminal-style card design

### 3. Skills & Tech Stack
- Grid of tech icons/badges (Python, TensorFlow, PyTorch, LangChain, OpenAI, Docker, etc.)
- Categorized: Languages, Frameworks, Tools, Cloud
- Animated progress bars or glowing card hover effects

### 4. Projects
- Card grid showcasing 4 placeholder AI/ML projects
- Each card: title, description, tech tags, GitHub/demo links
- Hover effect with glow/border animation

### 5. Contact
- Simple contact form (name, email, message)
- Social links (GitHub, LinkedIn, Twitter/X)
- Gradient CTA section
